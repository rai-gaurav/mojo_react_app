# Create charts with Perl(Mojolicious) and amCharts

## Requirements

1. [Mojolicious](https://mojolicious.org/)
2. [amCharts](https://www.amcharts.com/)
3. [React.js](https://reactjs.org/)

### How to run

```
cd mojo_app
morbo ./script/mojo_app
```

or on **Windows**

```
cd mojo_app
morbo.bat .\script\mojo_app
```
